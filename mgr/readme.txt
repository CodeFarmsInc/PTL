                ----------------------------------
                     PATTERN TEMPLATE LIBRARY
                ----------------------------------
                         directory mgr

This directory contains the source code of the Template Manager (mgr.cpp),
and the source for a special version of diff (diff.cpp) which compares
two files, and in the case of disagreement prints only a short
message about the size of the files, and the first line which disagrees.
In order to recompile programs in this directory, type:  all
